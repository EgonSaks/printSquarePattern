# Print Square Pattern in Go

### Go program prints hollow rectangle pattern in Go of N rows and M columns. 

## squareA

### Pattern is formed by using 4 different symbols:
````
 "0", "-", "|", " " 
````
#### Example

#### Input:

#### Number of rows: 10 
#### Number of columns: 10

#### Output:

`````
% go run test/main.go

o--------o
|        |
|        |
|        |
|        |
|        |
|        |
|        |
|        |
o--------o
`````

## squareB

### Pattern is formed by using 4 different symbols:
````
 "/", "\", "*", " " 
````
#### Example

#### Input:

#### Number of rows: 10 
#### Number of columns: 10

#### Output:

`````
% go run test/main.go

/********\
*        *
*        *
*        *
*        *
*        *
*        *
*        *
*        *
\********/

`````
